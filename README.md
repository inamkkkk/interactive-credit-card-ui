# Interactive Credit Card UI

A Flutter project showcasing an interactive credit card input form with a dynamically updated card UI.

## Features

- Real-time credit card visual updates as the user types.
- Form validation for card number, expiry date, and CVV.

## Dependencies

- provider: For state management.
- intl: For date formatting.
- mask_text_input_formatter: For input masking.

## Getting Started

1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to start the application.
